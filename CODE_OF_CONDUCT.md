# Code of Conduct · 行为准则

## 我们的承诺

InkForge 是一个开放协作的开源项目。为了让每位参与者——无论经验、年龄、身体状况、可见或不可见的差异、性别认同与表达、性取向、教育背景、社会经济地位、国籍、相貌、种族、宗教信仰——都能在没有骚扰的环境里贡献代码与想法，我们承诺：

- 用同理心和善意对待他人
- 尊重不同的观点、经验和使用场景（写小说的、做研究的、做工具的，需求很不一样）
- 接受建设性批评，并以同样的态度给出批评
- 关注对**项目和社区最有利**的事，而不是个人胜负
- 对自己的言行负责，犯错时承认、道歉、修正

## 不可接受的行为包括但不限于

- 使用性化语言或图像，以及任何形式的性关注与挑逗
- 嘲讽、侮辱、歧视性评论、人身攻击
- 公开或私下骚扰
- 未经允许公开他人的私人信息（住址、邮箱等"人肉"）
- 其他在专业场景下被合理视为不当的行为

## 适用范围

本准则适用于所有项目空间——Issue、PR、Discussion、Discord/Slack（如有）、邮件列表、线下活动——以及在这些空间之外**代表项目**的场合。

## 执行

违反行为可通过以下方式举报：

- GitHub 的 [Report content](https://docs.github.com/en/communities/maintaining-your-safety-on-github/reporting-abuse-or-spam) 功能
- 邮件至 `conduct@inkforge.app`（仅维护者收到，会保密）

维护者会在 7 天内回应。可能的处理包括：

1. **私下提醒**——告知违反点，给出整改建议
2. **公开警告**——在 Issue/PR 中明确指出
3. **临时封禁**——一段时间内禁止互动
4. **永久封禁**——严重或反复违规

申诉机制：被处理者可在 30 天内邮件申诉，由其他维护者复核。

## 来源

本准则改写自 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)。

> 简而言之：写小说和写代码都需要安静的环境，请把这里当作一个写作者的图书馆来对待。
